gentic AI Video Onboarding System

AI-powered onboarding and verification platform built using Flask, Groq AI, HTML, CSS, and JavaScript.
This project provides an interactive AI-based onboarding experience with live communication, document verification, and dynamic user interaction.

Features
AI-powered onboarding assistant
Real-time conversational interaction
Speech-to-text integration
Text-to-speech AI responses
Camera preview support
Microphone toggle functionality
Dynamic onboarding workflow
Role-based onboarding flow
Resume and document upload system
Verification dashboard
Modern responsive UI

Installation
1. Clone Repository
git clone https://github.com/yourusername/agentic-ai-video-onboarding-system.git
2. Navigate to Project Folder
cd agentic-ai-video-onboarding-system
3. Install Dependencies
pip install -r requirements.txt
4. Add Environment Variables

Create a .env file and add:

GROQ_API_KEY=your_api_key
5. Run Application
python app.py
